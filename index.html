Projects

Smart Gloves for Mute people

This project presents the design and implementation of a Smart Glove system utilizing the ESP32 microcontroller for the translation of sign language gestures into spoken and written language. The ESP32, renowned for its versatility and connectivity capabilities, serves as the core component for gesture recognition, data processing, and communication with external devices. The glove integrates flex sensors and accelerometers to capture hand movements, which are then transmitted to the ESP32 for real-time interpretation using machine learning algorithms. The processed data is subsequently converted into textual or auditory outputs, enabling seamless communication between sign language users and non-signers. Through the utilization of the ESP32 platform, this project endeavours to create an affordable, portable, and accessible solution to bridge the communication gap for the deaf and dumb community, fostering inclusivity and interaction in diverse social settings.
